# read intermediate data for the given experiment and step to qs2 file

read intermediate data for the given experiment and step to qs2 file

## Usage

``` r
read_intermediate_data(path, step, experiment)
```

## Arguments

- path:

  string with the input directory of the qs2 file

- step, :

  string with the step name

- experiment:

  string with the experiment name
