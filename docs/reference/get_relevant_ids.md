# Function to get relevant identifiers from the environment

Function to get relevant identifiers from the environment

## Usage

``` r
get_relevant_ids(identifiers, dt)
```

## Arguments

- identifiers:

  A character vector of identifier names to fetch from the environment

- dt:

  A data.table containing the columns to be checked against the
  identifiers

## Value

A character vector of relevant identifiers that are present in the
data.table
