# Detect model of data in data.table

Detect model of data in data.table

## Usage

``` r
# S3 method for class 'data.table'
data_model(x)
```

## Arguments

- x:

  data.table of raw drug response data containing both treated and
  untreated values.

## Value

string with the information of the raw data follows single-agent or
combination data model
